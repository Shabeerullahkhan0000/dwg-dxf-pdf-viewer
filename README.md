# DWG DXF PDF Viewer

## Project Overview  
DWG DXF PDF Viewer is a web-based application that allows users to view DWG, DXF, and PDF files seamlessly in their browsers. It provides an intuitive interface for users to interact with their drawings and documents without needing specialized software.

## Features  
- **File Support**: Supports viewing DWG, DXF, and PDF file formats.  
- **User-Friendly Interface**: Clean and easy-to-use interface designed for all types of users.  
- **Zoom & Pan**: Users can zoom in/out and pan across the document for detailed examination.  
- **Subscription Plans**: Offers various subscription plans for users with different needs.  
- **Performance Optimization**: Fast loading times and smooth performance across devices.  
- **Cross-Platform**: Compatible with all major web browsers and platforms. 

## Subscription Plans  
1. **Free Plan**:  
   - Basic features  
   - Limited file size  

2. **Pro Plan**:  
   - Access to all features  
   - Increased file size limit  
   - Monthly subscription fee  

3. **Enterprise Plan**:  
   - All Pro features  
   - Custom solutions for businesses  
   - Dedicated support  
   - Contact for pricing  

## Technical Requirements  
- **Frontend**:  
  - HTML5, CSS3, JavaScript  
  - Responsive design for mobile and desktop  
- **Backend**:  
  - Node.js or any preferred backend language  
- **Database**:  
  - MongoDB, PostgreSQL, or MySQL  

## Pages  
- **Home Page**: Introduction and overview of the application.  
- **Upload Page**: Users can upload files to view.  
- **Viewer Page**: Main page to view and interact with documents.  
- **Subscription Page**: Information on subscription plans and registration.  
- **Contact Page**: Help and support contact information.
